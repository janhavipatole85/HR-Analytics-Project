# HR-Analytics-Project
HR Analytics project using Excel, SQL, Power BI and Tableau to analyze employee attrition and workforce trends.
## Project Overview
This HR Analytics project focuses on analyzing employee attrition, job satisfaction, salary trends, and workforce performance using data analytics tools.

The goal of this project is to identify key factors affecting employee retention and help organizations make data-driven HR decisions.

---

## Tools Used
- Excel (Data Cleaning & Preparation)
- SQL (Data Analysis Queries)
- Power BI (Interactive Dashboard)
- Tableau (Data Visualization)

---

## Dataset Information
The dataset contains employee details such as:
- Employee ID
- Department
- Job Role
- Salary
- Gender
- Age
- Attrition
- Job Satisfaction
- Work Experience

---

## Key Analysis Performed
- Employee Attrition Analysis
- Department-wise Employee Count
- Salary Analysis
- Gender Distribution
- Job Satisfaction Analysis
- Employee Performance Trends

---

## SQL Queries Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- COUNT
- AVG
- JOIN

---

## Dashboard Features
- Interactive Filters
- Attrition Rate Visualization
- Department-wise Analysis
- Salary Insights
- Employee Distribution Charts

---

## Key Insights
- Higher attrition observed in Sales department
- Employees with lower salaries showed higher attrition
- Job satisfaction impacts employee retention
- Majority employees belong to Technology and Sales departments

---

## Project Outcome
This project helps HR teams understand workforce trends and improve employee retention strategies using data analytics.
